# Папка secrets/

Эта папка содержит чувствительные данные и НЕ должна попадать в git репозиторий.

## Структура:

- `user_data/` - данные пользователей (договоры, счета и т.д.)
- `vk/` - файлы с API токенами и паролями VK.ОРД (НЕ коммитить!)
- `vk_ord_tokens.json` - токены пользователей VK.ОРД (игнорируется git)
- `vk_ord_state.json` - состояние VK.ОРД (игнорируется git)
- `*.key` - ключи шифрования (если используются)

## Важно:

- Все файлы в этой папке автоматически игнорируются git
- Не добавляйте сюда файлы с реальными токенами, паролями или email адресами в репозиторий
- При клонировании репозитория создайте необходимые файлы вручную на основе примеров
